# carousel-app

## Project setup
```
npm install
```
### Ensure mock api is running using json-server before compiling the application
```
json-server --watch mock-api.json
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
